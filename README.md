
### About

Ethereum Wallet to understand how does a transaction occur, how does mnemonic work, and how do accounts get created. 

### How to Start

* git clone https://github.com/bansaltushar014/Ethereum_Wallet.git
* Start Ganache
* npm install
* npm start


### Technology & Tools

* Web3
* React
* Ganache
